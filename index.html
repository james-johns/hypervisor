Hypervisor for ARM systems.

A simple, bespoke hypervisor for ARM systems supporting hardware virtualisation extensions. Currently only targeting the Cubietruck (Cubieboard 3) ARM board, featuring an Allwinner A20 SoC (dual core Cortex-A7).

Repository

To check out the project, run the following:

$ git clone https://github.com/james-johns/hypervisor/

Branching model

The branching model used by this repository is similar to the 'successful git branching model' presented by nvie (http://nvie.com/posts/a-successful-git-branching-model/).

master - Represents the current stable release. Releases are tagged as appropriate. development - Represents the current project state in development. This is the only branch to be merged into master feature/\* - A feature currently in development. Feature branches are branched from and merged back into the development branch.
